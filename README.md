# AI DDR Report Generator

This system reads:

1. Inspection Report
2. Thermal Imaging Report

It automatically generates a structured DDR (Detailed Diagnostic Report).

## Features

- Extracts observations from inspection reports
- Reads thermal image findings
- Combines both sources
- Generates a client-friendly DDR report
- Handles missing information

## Output Structure

1. Property Issue Summary
2. Area-wise Observations
3. Probable Root Cause
4. Severity Assessment
5. Recommended Actions
6. Additional Notes
7. Missing Information
